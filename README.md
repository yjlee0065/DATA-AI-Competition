# [Project] 빵타스틱: 성심당 방문객 행선지 추천 시스템
> **소비 패턴 및 유동인구 분석 기반 대전 중구 지역 경제 활성화 솔루션**

본 프로젝트는 대전의 랜드마크인 '성심당'에 집중된 관광객을 대전 중구 전역으로 확산시키기 위해 기획되었습니다. 신한카드 소비 데이터와 유동인구 데이터를 분석하여, 방문객의 특성에 맞는 최적의 관광 코스를 제안함으로써 지역 경제의 선순환 구조를 만드는 것을 목적으로 합니다.

---

## 1. 프로젝트 개요
* **분석 대상**: 성심당 방문객 및 대전 중구 내 주요 관광지/상권
* **데이터 규모**: 신한카드 소비 데이터(Cell ID별 유입 고객), 대전 중구 유동인구 데이터, 소셜 크롤링 데이터
* **핵심 분석**:
    - 성심당 이용 고객의 유입/유출 이동 경로 파악
    - 카드 소비 패턴 분석을 통한 방문객 페르소나 도출
    - K-Means 군집화를 통한 테마별 관광지 분류 및 추천 알고리즘 구현

## 2. 사용 데이터 및 기술 스택
* **사용 데이터 (Data)**: 
    - **카드 데이터**: 신한카드 Cell ID별/지역별 업종 유입 고객 소비 데이터
    - **유동인구 데이터**: 대전 중구 지역별 시간대/연령별 유동인구 데이터
    - **관광 정보**: 공공데이터포털 및 소셜 미디어(블로그, 인스타그램) 크롤링 데이터
* **기술 스택 (Tech Stack)**:
    - **Language**: `Python`
    - **Environment**: `Jupyter Notebook`
    - **Library**:
        - **Analysis**: `Pandas`, `Numpy`, `Scikit-learn (K-Means)`
        - **Visualization**: `Matplotlib`, `Seaborn`, `Folium` (지리 정보 시각화)
        - **Crawling**: `Selenium`, `BeautifulSoup4`

## 3. 주요 분석 프로세스
### 1) 데이터 전처리 및 분석 (Data Acquisition & Cleaning)
- **카드/유동인구 데이터 정제**: 대전 중구 지역의 Cell ID별 데이터를 분석 목적에 맞게 병합 및 가공
- **관광지 정보 수집**: 공공데이터 및 소셜 미디어 크롤링을 통해 대전 중구 내 주요 명소 데이터셋 구축
- **소비 패턴 필터링**: 성심당 방문 전후 3시간 이내의 소비 기록을 추출하여 실제 관광객의 이동 경로 특정

### 2) 데이터 분석 기법
- **군집 분석 (Clustering)**: K-Means 알고리즘을 활용해 관광지를 5개 테마(공연, 맛집, 전시, 키즈, 역사)로 분류
- **협업 필터링 기반 추천**: 사용자의 성별, 연령대, 방문 시간대를 입력값으로 하여 가장 적합한 군집의 행선지 추천
- **공간 분석**: 성심당과의 물리적 거리와 유동인구 흐름을 고려한 최적 동선 도출

## 4. 핵심 인사이트 (성심당 방문객 중심)
- **이동 경로의 편중**: 분석 결과 성심당 방문객의 대다수가 특정 구역에만 머물다 떠나는 '낙수효과 부족' 현상 확인
- **잠재적 니즈 발견**: 연령대별로 선호하는 테마(예: 20대-전시/맛집, 가족단위-키즈/역사)가 뚜렷하게 구분됨을 데이터로 증명
- **현장 접점 솔루션**: 성심당 대기줄에서의 긴 대기 시간을 활용하여 관광 정보를 제공하는 **'QR 코드 기반 추천 서비스'** 시나리오 설계
